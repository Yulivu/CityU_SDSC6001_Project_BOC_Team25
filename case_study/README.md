## Case study (MA-GNN aspect weights)

This folder contains scripts to generate the case study artifacts described in `CASE_STUDY_GUIDE.md`.

### Outputs

- `candidate_users.json`: candidate users with stable vs unstable aspect variance contrast
- `case_study_raw.json`: per-user aspect weights + MA-GNN Top-10 + popularity Top-10 + ground truth ranks
- `best_case_user.json`: a single selected user from `case_study_raw.json`

### Run

Prerequisites:

- `outputs/*.parquet` generated by the data prep pipeline
- a trained MA-GNN checkpoint (default: `artifacts/magnn_full_cpu_10e_es.pt`)

```bash
python case_study/find_case_users.py
python case_study/run_case_study.py --ckpt artifacts/magnn_full_cpu_10e_es.pt --device cpu
python case_study/pick_best_case.py
```
